# TinyMe Issue Tracker

This repository is for bug reports, issues, and feedback for TinyMe only.

## Before opening an issue
Please include the following details so we can investigate quickly:

- **Game version** (Minecraft version)
- **Mod version** (TinyMe version)
- **Steps to reproduce** (clear, numbered steps)
- **Logs** (crash reports/latest.log) if available

## Labels used in this tracker
- `bug`
- `feature`
- `invalid`
- `duplicate`
- `question`

## Important
Do not post unrelated content in this repository.
